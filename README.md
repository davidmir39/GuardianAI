# GuardianAI
Un servicio de IA que procesa transacciones bancarias en tiempo real y devuelve una probabilidad de fraude. Si la probabilidad es alta, la transacción se bloquea o se envía a revisión manual

El dataset utilizado es el archivo Base.csv de este enlace https://www.kaggle.com/datasets/sgpjesus/bank-account-fraud-dataset-neurips-2022?select=Base.csv
